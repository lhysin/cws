cws.io
===================
### item-apiserver
#### by srpingboot 3.0.5 with kotlin


Run with Gradle (default profile 'default')
-------------------
```
# Windows
gradlew.bat clean build && gradlew.bat :bootRun

# Other (Linux, macOs)
./gradlew clean build && ./gradlew :bootRun
```

You can then access
-------------------
http://localhost:8080/swagger-ui/index.html
